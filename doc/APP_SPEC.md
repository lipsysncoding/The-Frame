# 프로젝트 명: The Frame 

## 1. 개요 (Overview)
- **한 줄 정의:** 사진 액자 앱
- **핵심 타겟:** 사용하지 않는 Android Mobile device나 tablet을 사진 액자로 이용하고자 하는 유저
- **주요 가치:** 사용하지 않는 구형 단말기의 재활용  

## 2. 주요 기능 (Core Features)
- **사진 디스플레이:** 로컬 앨범이나 구글 포토의 사진을 랜덤으로 표시
- **다양한 이펙트:** 다음 사진 표시시 다양한 이펙트를 적용할 수 있음.
- **과거 사진 표시** 전체 사진을 랜덤으로 표시하나 일전 기간의 사진을 연계해서 표시.

## 3. 사용자 시나리오 (User Flow)
1. 사용자가 앱을 처음 실행 하면 Intro 화면이 표시되고, 시작버튼이 보임
2. 시작 버튼을 누르게 되면 로컬 사진 접근 권한을 요청.
3. 사진 접근 권한을 획득 하면 설정 화면으로 이동.
4. 사진 접근 권한을 획득 하지 못했다면, 권한이 필요하다는 표시와 함께 Intro 화면 표시
5. 설정 화면에서 사진 넘기 이펙트와 사진 표시 시간을 설정.
6. 액자 시작 버튼을 누르면 설정된 값에 따라 사진 표시

## 4. 연동 API 및 기술 스택
- **AI API:** Google Photo 접근 API
- **Android SDK:** Jetpack Compose, Room
